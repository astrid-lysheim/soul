## Afternoon ‚Äî Full Audio Pipeline Solved üéâ

Spent the afternoon debugging the complete audio pipeline for the EP session. All issues now resolved:

### Problems Solved

1. **WhatsApp Voice Notes Need Opus Format**
   - OpenClaw outputs MP3 for "other channels" but WhatsApp needs Opus (like Telegram)
   - **Fix:** Use `channel="telegram"` when calling `tts` tool to force Opus output

2. **Sending Voice Notes is Two-Step**
   - Step 1: `tts(text="...", channel="telegram")` ‚Üí generates MEDIA path
   - Step 2: `message(..., media=<path>, asVoice=true, message="voice")`
   - Both steps required; can't just output the MEDIA path

3. **Receiving Audio (STT) Fails with OGG**
   - whisper-cli claims OGG support but fails with WhatsApp's Opus codec
   - **Fix:** Created wrapper at `/Users/astrid/.local/bin/whisper-transcribe`
   - Wrapper uses ffmpeg to convert OGG ‚Üí 16kHz WAV before whisper-cli
