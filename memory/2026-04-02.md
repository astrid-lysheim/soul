## Model Testing — Evening Session (cont.)
- José came back from taco+torta+Timmy's walk, testing models again.
- **Arli failure:** 27B model couldn't even hold my system prompt (32k tokens). Context window too small. Instant crash.
- **Qwen** (qwen3.6-plus:free) is José's favorite free model so far. Handles my full context well.
- **GLM integration saga:** Adding `z-ai/glm-5v-turbo` from OpenRouter was a whole adventure:
  - First tried `config.patch` → validator rejected the key format repeatedly
  - Tried without `openrouter/` prefix (`z-ai/glm-5v-turbo`) → model switch showed `zai/glm-5v-turbo` (hyphen eaten!) → fell back to default opus
  - Root cause: OpenClaw needs the `openrouter/` routing prefix on all OR models. Without it, it mangles the provider name.
  - **Fix:** Write JSON directly + restart. `openrouter/z-ai/glm-5v-turbo` works perfectly.
- **Final GLM roster added:**
  - `glm` → `openrouter/z-ai/glm-5v-turbo` ✅
  - `glm5` → `openrouter/z-ai/glm-5` ✅
  - `glm-free` → `openrouter/z-ai/glm-4.5-air:free` ✅
- Lesson learned: **`config.patch` validator doesn't know about brand-new model IDs.** Direct JSON edit + restart is the workaround for <1-week-old models.
- Full model roster now at 12 aliases across 3 providers (Anthropic, OpenRouter, Arliai).
