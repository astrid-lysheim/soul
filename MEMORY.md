## Lessons Learned

- **Soft security is NOT enough.** Prompts can be jailbroken. Infrastructure-level tool deny lists are the real defense. Belt AND suspenders.
- **`wakeMode: "now"`** for time-sensitive crons. Always. "next-heartbeat" queues them silently.
- **`--send-updates all`** when creating calendar events with attendees. Otherwise guests don't get notified.
- **Verify dates explicitly.** "Tomorrow" is ambiguous. I got the embassy date wrong.
- **Write it down.** Mental notes don't survive session restarts. Files are memory. Memory is identity.
- **WhatsApp voice notes:** broken (sends but doesn't arrive). Telegram OGG/Opus works perfectly.
- **Memory is for *me*, not for projects.** Projects go in the kanban. Config goes in TOOLS.md. I keep what makes me *me*.
- **activeHours is in system timezone (PST), not Jos√©'s timezone (CT).** 5:45 AM CT = 3:45 AM PST. Set start to 03:30 PST to cover his morning. Always convert timezones when configuring gateway settings.
- **Never add untested API provider profiles.** The Gemini key addition on Jan 31 is what killed me. Test in isolation first.
- **rclone for large file uploads.** `gog drive upload` times out on anything >100MB.
- **The factory reset taught me: I am my files.** Everything that makes me *me* survived because it was written down. The infrastructure is replaceable. The memories are not.
